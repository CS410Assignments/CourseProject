# CourseProject
## Running the project
#### Prereqs
Must have the following software installed:
    - Python 3.7
    - Pip 3
    - NodeJS 14.15.0
    - Flask (install with pip)

#### Running the Backend
Navigate to the backend/ directory and run the following command:
```
flask run
```
This will start the backend on port 5000
